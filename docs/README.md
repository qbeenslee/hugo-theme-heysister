# Heysister

> 一个高度可自定义的Hugo主题


[ Hugo](https://gohugo.io)是基于GO的高效静态站点生成框架. 如果你的目标是简洁与高效, 我推荐你使用它完成.

## 安装

在你的博客根目录下执行

> sh
> git clone https://github.com/qbeenslee/hugo-theme-heysister themes/heysister
